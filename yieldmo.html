Question 2:

Differences between JavaScript and TypeScript:

- TypeScript is an Object Oriented Programming Language while JavaScript is a Scripting Language

- TypeScript supports static typing while JavaScript can only support dynamic typing

Depending on the nature of my project, I might have different decisions to choose TypeScript or JavaScript. At its core, TypeScript is a strongly typed JavaScript that better supports OOP. Using TypeScript will prevent the complexities and confusions that the loosely typed JavaScript can create. If the project I have to work on has a large scale with great complexity, I’ll prefer TypeScript. However, if it’s just a small or maybe medium size project, I’ll just use JavaScript. Since the browser can’t directly read TypeScript, TypeScript will take some time to be compiled and converted to JavaScript. In conclusion, if my project is small and simple enough, I’ll prefer JavaScript. Otherwise, I’ll prefer TypeScript for larger scale and complicated projects.

Out of many challenging projects I’ve worked on, the one I’m most proud of is the scraping tool Chrome Extension. Back in my time with Secomus Technology, I was asked to work on a Chrome Extension that can crawl reviews from the major ecommerce platform AliExpress. The problem was, with a Chrome Extension, I couldn’t just install some handy npm libraries to aid me with scraping and processing the data. But I couldn’t do normal text processing, either. Since that would create so much unnecessary work and probably slow our extension down by a lot creating terrible experiences for our users.

Yet I didn’t let those obstacles prevent me from finishing the tasks. Using browserify, I was able to translate those handy npm libraries for scraping, processing data to a format that Chrome Extension can handle. I even utilized mass requests sending with asynchronous coding to create the fastest scraping experience. This Extension helped our team out-perform every other opponents in the market and created a whole new selling point for us
